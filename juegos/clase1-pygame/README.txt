INSTRUCCIONES PARA INSTALAR PIP Y PYGAME

1. Instalar pip (si no lo tienes):
   - En Windows, abre PowerShell o CMD y ejecuta:
     python -m ensurepip --upgrade
   - Si no funciona, descarga get-pip.py desde https://bootstrap.pypa.io/get-pip.py y ejec√∫talo:
     python get-pip.py

2. Instalar pygame:
   - Ejecuta en la terminal:
     pip install pygame

3. Ejecutar el ejemplo:
   - Navega a la carpeta clase1-pygame y ejecuta:
     python main.py

Si tienes varios Python instalados, puede que debas usar python3 y pip3 en vez de python y pip.
